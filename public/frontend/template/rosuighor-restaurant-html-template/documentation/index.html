Getting Started

Unzip the Rosuighor. There are all the template files in WWW folder. You can open the following files for viewing in browser.

  1. assets/css — folder with css files.
  2. assets/fonts — folder with icon fonts.
  3. assets/img — folder with image files.
  4. assets/js — folder with Javascript files.
  5. demo.html — page with the list of template pages.
  6. index.html, index2.html, index3.html, index4.html, ... — different home page demos.
  7. blog-*.html — blog pages.
  8. single-blog*.html — Single blog pages.
  9. about*.html — About pages.
  10. reservation.html — reservation page.
  11. gallery.html — Food gallery page.
  12. elements-*.html — shortcodes pages.
  13. menu.html — All types of menus page .
  14. single-category.html — Single type menus page.
  15. single-menu.html — Single Menu details page.
  16. contact.html — file with contact form and Google Map.

Upload the template files (inside the folder www) to server with the help of one of the FTP-clients (for example, Filezilla).

Section With Fixed Background image

To make the background image static parallex fixed add the class /.bg-fixed to section.
